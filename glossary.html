Glossary

HTML Links - Hyperlinks
HTML links are hyperlinks. You can click on a link and jump to another document.

HTML Lists
HTML lists allow web developers to group a set of related items in lists.

Unordered HTML List
An unordered list starts with the
    tag. Each list item starts with the
  • tag.
    The list items will be marked with bullets (small black circles) by default.

Ordered HTML List
An ordered list starts with the
    tag. Each list item starts with the
  1. tag. The list items will be marked with numbers by default.
The Description List Element
The HTML
element represents a description list.
The element encloses a list of groups of terms (specified using the
element) and descriptions (provided by
elements). Common uses for this element are to implement a glossary or to display metadata (a list of key-value pairs).
